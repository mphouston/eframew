Package org.simplemes.mes.demand.domain

Package org.simplemes.mes.demand.domain Description

Domain classes for the demand package. These classes define demand for production within the shop floor. This includes key elements such as Orders and Lot/Serial Numbers (LSN).

  • Class Summary 
    Class Description
    LSN Defines a lot/serial number (LSN) for tracking and identification of demand.
    LSNOperState This object holds the work state of an operation for an LSN.

    Order This is a manufacturing order that is used to produce a product on the shop floor.
    OrderOperState This object holds the work state of an operation for an order.
    OrderOperation Defines a routing for a specific order and the status of the operations for the order.
    OrderSequence Defines the sequence used to generate new order numbers.
