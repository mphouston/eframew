class LSNOperState
extends java.lang.Object
implements WorkStateTrait, WorkableInterface

      This object holds the work state of an operation for an LSN.

      This Work State contains the quantities in queue, work and done. This object also implements the WorkableInterface to encapsulate the process of working on various objects.
