Package org.simplemes.mes.demand

Package org.simplemes.mes.demand Description

Demand definition classes. These classes define demand for production within the shop floor. This includes key elements such as Orders and Lot/Serial Numbers (LSN).
  • Interface Summary 
    Interface Description
    DemandObject A Marker interface to flag an Order or LSN class.
    WorkableInterface This interface represents the smallest possible unit that can be worked.
