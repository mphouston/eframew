      • ScanResponseInterface postScan(ScanResponseInterface coreResponse, ScanRequestInterface scanRequest)

        Post-scan extension method for the scan() method. See the core method ScanService#scan(org.simplemes.mes.system.ScanRequestInterface) for details.

        Note: This method is part of the Stable API.

        Parameters:
        coreResponse - The response from the core method (or from other extensions that provide a response).
        scanRequest - The details of the request .
        Returns:
        Return a new response if your extension needs to return a different value. Return null to use core response. You can also modify the coreResponse if it is a POGO that allows modification.
