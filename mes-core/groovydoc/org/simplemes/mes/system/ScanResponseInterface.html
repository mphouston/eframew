Package: org.simplemes.mes.system

[Groovy] Interface ScanResponseInterface


  • interface ScanResponseInterface
extends org.simplemes.eframe.web.undo.UndoableInterface

    The response for the scan() method. This scan resolution is typically handled by the ScanService. This POGO is returned after the scan has been evaluated.
