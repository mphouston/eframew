        • groovy.lang.Tuple2<java.lang.Integer, java.lang.Integer> calculateFromAndSizeForList(java.util.Map params, boolean allowNull)

          Calculates the effective from (page start) and size(page size) for standard list queries. Supports GUI toolkit-style start and count. Limits the size to 100 (configurable) for safety.

          Typical usage in a controller: 

           def (int from, int size) = ControllerUtils.calculateFromAndSizeForList(params)
          Parameters:
          params - The request parameters. Integer and String values are supported.
          allowNull - If true, then the returned values can be null. Only use this option if you have fallback's for max in your code.
          Returns:
          A Tuple with the effective offset and max. Never null, always Integers. (Defaults: 0,UIDefaults.PAGE_SIZE).

        • groovy.lang.Tuple2<java.lang.String, java.lang.String> calculateSortingForList(java.util.Map params)

          Calculates the effective sortdatafield and sortorder for standard list queries. Supports toolkit-style sort[field]=asc as well as the simpler sort and order parameters. The toolkit-style sort[field]=asc takes precedence over the simpler style.

          Typical usage in a controller: 

           def (sortField, direction) = ControllerUtils.calculateSortingForList(params)
          Parameters:
          params - The request parameters. String values are supported. The toolkit style takes precedence over the simple style.
          Returns:
          A Tuple with two values: the sort field and direction.
