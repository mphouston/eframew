Package: org.simplemes.eframe.controller

[Groovy] Class BaseController

    • org.simplemes.eframe.controller.BaseController

    • Provides the standard error handler definition for a normal controller. This sends a response from the controller with a JSON encoded MessageHolder as the response content. Note: This is a base class instead of the easier to use than a trait. For some reason, the groovy compiler fails with an error 'canonicalization NoSuchMethodException: java.lang.annotation.Annotation.equals()' during compilation with the trait. So, we clone this in the local project to avoid the issue. The problem is traced to the use of

      Error:
      in a trait. Groovy states that traits and AST transforms aren't guaranteed to work.

        • protected MessageHolder lastErrorMessage

          The most recent error message returned to the client. This is mainly used in unit tests since the Micronaut client throws an exception when a BAD_REQUEST status is returned. There is no way to detect the real exception message while in a unit test.
