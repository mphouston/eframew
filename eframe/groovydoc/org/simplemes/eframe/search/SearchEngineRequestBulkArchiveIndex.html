class SearchEngineRequestBulkArchiveIndex
extends java.lang.Object

      This encapsulates a bulk index request on a list of archive references. It primarily sends the index request to the external search engine and does minimal processing and error checking.

      Logging

      The logging for this class that can be enabled:
      • trace - When a bulk index fails, the entire contents are logged (up to 20k chars).
