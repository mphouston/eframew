      • org.simplemes.eframe.search.SearchableJacksonFilter 

      • class SearchableJacksonFilter
extends com.fasterxml.jackson.databind.ser.impl.SimpleBeanPropertyFilter

        Defines a filter that looks at the searchable static field for any excluded fields and removes them from the JSON for the search engine.
