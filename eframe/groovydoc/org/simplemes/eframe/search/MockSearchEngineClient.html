Package: org.simplemes.eframe.search

[Groovy] Class MockSearchEngineClient

    • org.simplemes.eframe.search.MockSearchEngineClient
    • All Implemented Interfaces and Traits:
      SearchEngineClientInterface

      @groovy.util.logging.Slf4j
class MockSearchEngineClient
extends java.lang.Object
implements SearchEngineClientInterface

      This mock client simulates the search interface to the external search engine (Elastic Search in this case). This is used in production when the search engine is not configured.
