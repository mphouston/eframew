Package org.simplemes.eframe.search

Package org.simplemes.eframe.search Description

Classes that provide the search engine features.
  • Interface Summary 
    Interface Description
    AdjustQueryInterface Defines the API for the adjustQuery extension point.
    SearchEngineClientInterface This interface defines the API provided by the search interface to the external search engine (Elastic Search in this case).
    SearchEngineRequestInterface This is the common interface for all request handled by the search engine clients.
  • Class Summary 
    Class Description
    ArchiveSearchHit A single archive search hit from the search engine.
    MockRestClient This is a mock client that can simulate the external search engine's response to various requests.
    MockRestClient.MockResponse A mock for the response from the rest client.
    MockSearchEngineClient This mock client simulates the search interface to the external search engine (Elastic Search in this case).
    MockSearchEngineRequest A mock request used for testing.
    PassAllJacksonFilter Defines a filter that allows all fields to be serialized.
    SearchDomainSettings Defines the search settings for single domain class.
    SearchEngineClient This helper encapsulates the search interface to the external search engine (Elastic Search in this case).
    SearchEnginePoolExecutor The thread pool executor that provides background processing of search engine requests.
    SearchEnginePoolExecutor.SearchThreadFactory The thread factory for the pool.
    SearchEngineRequestBulkArchiveIndex This encapsulates a bulk index request on a list of archive references.
    SearchEngineRequestBulkIndex This encapsulates a bulk index request on a list of domain objects.
    SearchEngineRequestIndexObject This encapsulates a index request on a domain object.
    SearchEngineRequestObjectRemoval This encapsulates a request to remove a domain object from the search engine index.
    SearchEngineRequestQueue The search request queue.
    SearchHelper This helper encapsulates the search interface.
    SearchHit A single search hit from the search engine.
    SearchResult This class contains the results of a search.
    SearchStatus This class contains the current search engine status.
    SearchTransactionEventListener Listens for the transaction commit events on domain objects.
    SearchValidation Provides validations for Domain Entities that check for search mis-configurations.
    SearchableJacksonFilter Defines a filter that looks at the searchable static field for any excluded fields and removes them from the JSON for the search engine.
