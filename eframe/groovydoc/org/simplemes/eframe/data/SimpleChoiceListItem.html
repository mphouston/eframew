    • All Implemented Interfaces and Traits:
      ChoiceListItemInterface

      @groovy.transform.ToString(includePackage: false, includeNames: true)
class SimpleChoiceListItem
extends java.lang.Object
implements ChoiceListItemInterface

      A generic entry of the ChoiceListItemInterface element. This uses a static display value for the current locale. Not localization is performed by this class.

      Note: This class should not be cache between HTTP requests. It should be generated for the request and discarded when the request is done.
