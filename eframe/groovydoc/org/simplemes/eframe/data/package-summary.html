Package org.simplemes.eframe.data

Package org.simplemes.eframe.data Description

Defines data fields and classes for manipulating the data.
  • Interface Summary 
    Interface Description
    ChoiceListInterface Defines the list of possible choices for a combobox type widget.
    ChoiceListItemInterface Defines a single instance of the possible choices for a class.
    ConfigurableTypeInterface Defines an open-ended data type that can be configured by the end-user without extensive GUI or domain changes.
    EncodedTypeInterface Flags a basic type that is stored as an encoded value in the database.
    FieldDefinitionInterface Defines fields used by the framework.
  • Class Summary 
    Class Description
    AdditionFieldDefinition A field definition for a field added by a module addition.
    CustomFieldDefinition A field definition for custom fields created by user or addition modules.
    EncodedTypeConverters Convert to/from SQL String to encoded types.
    EncodedTypeListUtils Provides utility methods to access a list of choices that implement the ChoiceListInterface.
    FieldDefinitionFactory Builds the correct field definition for a given property of class field.
    FieldDefinitions Contains all of the fields defined for domain/POGO object.
    SimpleChoiceListItem A generic entry of the ChoiceListItemInterface element.
    SimpleFieldDefinition A simple field definition.
