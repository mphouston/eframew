Package: org.simplemes.eframe.data.format

[Groovy] Interface FieldFormatInterface


  • interface FieldFormatInterface

    Defines the basic field format handling. Implementations are provided for all of the major field types (e.g. String, BigDecimal, etc).

    Note: The FieldDefinitionInterface passed into the format/parse/etc classes is used to provide additional information needed to process the raw value. This is only used for more complex types. Simple types like BigDecimal and Integer don't use the field definition.
