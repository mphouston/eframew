[Groovy] Class ConfigurableTypeDomainFormat

        • org.simplemes.eframe.data.format.ConfigurableTypeDomainFormat
        • All Implemented Interfaces and Traits:
          ConfigurableFormatInterface

          class ConfigurableTypeDomainFormat
extends DomainReferenceFieldFormat
implements ConfigurableFormatInterface

          Defines the format for a field that is Configurable Type used with a domain such as Flex Type. This is a open-ended list of dynamic custom fields that can be configured by the user.
