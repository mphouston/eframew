Package: org.simplemes.eframe.data

[Groovy] Interface ConfigurableTypeInterface

  • Defines an open-ended data type that can be configured by the end-user without extensive GUI or domain changes. This is used to indicate that a given domain property is a configurable type and can be used by the framework in the definition GUIs.

    This will typically be extended by a specific field type interface.

    See Configurable Types for details.

      • List determineInputFields(java.lang.String configurableTypeFieldName)

        Builds the fields needed to configure this type. This will be used to build the GUI field HTML elements.

        Parameters:
        configurableTypeFieldName - The name of the Configurable Type field that the caller wants the fields for. This is used to specify the prefix for the values in the custom field holder.
        Returns:
        The list of fields needed to configure this type. These fields should match the names and types of the data fields.

      • java.lang.String toStringLocalized(Locale locale)

        Parameters:

        Returns:
