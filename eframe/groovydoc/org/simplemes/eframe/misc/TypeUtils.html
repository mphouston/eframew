        • @groovy.transform.CompileDynamicCompileDynamic static java.lang.String toShortString(java.lang.Object object, java.lang.Boolean addTitle)

          A short string for the given object. Usage is similar to toString(), but returns just one key identifier. Suitable for drop-down lists, etc. Will call toShortString() on the object if it exists. If not, then will attempt to use the primary key (per DomainUtils.getPrimaryKeyField()). Finally, will fallback to toString() if none of these work.

          Parameters:
          object - The object. Domain object of POGO.
          addTitle - If true, then add the title to the short string (e.g. 'ABC (title)'). (Default: false).
          Returns:
          The short string representation of this (e.g. primary key value). Can be null.
