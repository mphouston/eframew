        • static java.lang.String formatLargeNumberReadable(long value, java.util.Locale locale)

          Build a human-readable large number with M, K, G, etc. 3,543,725 is displayed as '3.1M' and 54,543,725 is displayed as '54M'. Makes sure at least 2 significant digits are displayed if the whole number is <10. Supports K (1000), M, M, G, T (tera). This method does not round to nearest whole number and uses the 'normal' definition of kilo, mega, etc (K=1000,M=1000000, etc).

          Parameters:
          value - The value to display the shortened readable number.
          locale - The locale to use for display (Default: Request Locale, or Locale.default).
          Returns:
          The string. (e.g. 3,543,725 is returned as '3.1M').
