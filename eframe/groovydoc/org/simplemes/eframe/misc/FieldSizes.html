Package: org.simplemes.eframe.misc

[Groovy] Class FieldSizes

    • org.simplemes.eframe.misc.FieldSizes 

    • class FieldSizes
extends java.lang.Object

      Defines many field size limits used in the framework. These fields can be used by your application, but should be referenced inside of your own FieldSizes \ class. For example: 

       package org.example.misc
 class FieldSizes {
   public static final int MAX_CODE_LENGTH = org.simplemes.eframe.misc.FieldSizes.MAX_CODE_LENGTH
 }
      Note: The package name org.simplemes.eframe.misc should be used if your class is named FieldSizes.
