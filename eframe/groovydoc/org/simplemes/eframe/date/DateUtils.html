        • static java.lang.String formatElapsedTime(long elapsedTimeMS, java.util.Locale locale, java.lang.String precision)

          Build a human-readable elapsed time string from the given elapsed time (ms), but only shows one number to make it readable. Use this in scenarios when super-precise display is not needed. Supports seconds, minutes, days and years. This method does not round to nearest whole number when precision is set to LOW (default).

          This method ignores leap years. All years are assumed to be 365 days.

          Parameters:
          elapsedTimeMS - The elapsed time (ms).
          locale - The locale to use for the elapsed time string (Default: Request Locale, or Locale.default).
          precision - The precision to display. (Default: PRECISION_LOW).
          Returns:
          The string. (e.g. for 70 seconds, returns "1 minute").

        • static java.lang.String formatRelativeTime(long relativeTime, java.util.Locale locale, java.lang.String precision)

          Build a human-readable relative time. Only shows one number to make it readable. Handles past times (negative) and future times ('2 minutes ago' and '2 minutes from now'). Use this in scenarios when super-precise display is not needed. Supports seconds, minutes, days and years. This method does not round to nearest whole number when precision is set to LOW (default).

          This method ignores leap years. All years are assumed to be 365 days.

          Parameters:
          relativeTime - The related time (ms).
          locale - The locale to use for the elapsed time string (Default: Request Locale, or Locale.default).
          precision - The precision to display. (Default: PRECISION_LOW).
          Returns:
          The string. (e.g. for 70 seconds, returns "1 minute ago").
