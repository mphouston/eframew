Package: org.simplemes.eframe.domain

[Groovy] Class DomainUtils

    • org.simplemes.eframe.domain.DomainUtils 

    • class DomainUtils
extends java.lang.Object

      Domain Object manipulation utilities. These class provide common domain class utilities that simplify access to domain objects and their use. This includes methods to return the primary keys for a domain object.
