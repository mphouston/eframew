[Java] Class ValidationError

    • org.simplemes.eframe.domain.validate.ValidationError
    • All Implemented Interfaces and Traits:
      ValidationErrorInterface

      public class ValidationError
extends java.lang.Object
implements ValidationErrorInterface

      A single validation error on a field. The display value (toString()) uses the messages.properties file to find the matching error string, with replaceable parameters. The key in the .properties file is "error.$code.message". The first argument ({0}) is the fieldName for this error.
