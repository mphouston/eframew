Package org.simplemes.eframe.domain

Package org.simplemes.eframe.domain Description

Classes to support access to Domain objects and their structure.
  • Interface Summary 
    Interface Description
    BaseRepository Defines common repository features.
  • Class Summary 
    Class Description
    DomainBinder Provides standardized binding of HTTP-style parameters to a domain or POGO class.
    DomainReference A simple POGO that contains the information needed for a reference to a domain property.
    DomainSaveTransactionEvent Event published when a single domain record is saved.
    DomainUtils Domain Object manipulation utilities.
    EFrameJdbcRepositoryOperations A substitute for the micronaut JdbcRepositoryOperations implementation to add features for the enterprise framework.
    EFrameJdbcRepositoryOperationsJava A substitute for the micronaut JdbcRepositoryOperations implementation to add features for the enterprise framework.
    PersistentProperty Defines a single persistent property for a given domain object.
    SQLUtils Common utility methods used to access SQL data directly.
  • Exception Summary 
    Exception Description
    OptimisticLockException This is triggered when an update fails due to optimistic lock failure (version mis-match or record deleted).
    UpdateFailedException Exception thrown when a single-record update fails in the JDBC framework.
