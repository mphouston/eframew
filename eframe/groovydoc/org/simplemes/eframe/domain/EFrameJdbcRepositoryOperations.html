Package: org.simplemes.eframe.domain

[Groovy] Class EFrameJdbcRepositoryOperations

        • org.simplemes.eframe.domain.EFrameJdbcRepositoryOperations

        • A substitute for the micronaut JdbcRepositoryOperations implementation to add features for the enterprise framework. Provides: Check for transactions on updates, support for optimistic locking and possible work-arounds for issues.

          Logging

          The logging for this class that can be enabled:
          • trace - Logs the queries that are checked for transaction.
