[Groovy] Class ReportResourceHandler

    • org.simplemes.eframe.reports.ReportResourceHandler
    • All Implemented Interfaces and Traits:
      net.sf.jasperreports.engine.export.HtmlResourceHandler

      class ReportResourceHandler
extends java.lang.Object
implements net.sf.jasperreports.engine.export.HtmlResourceHandler

      Handles the report engine's resource (image) processing suitable for use in a controller. This provides the correct resource path/names using a convention for use the ReportController.image method. The external URI for the resources uses resource ID and the reportPath to make the reference unique.
