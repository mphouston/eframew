Package: org.simplemes.eframe.reports

[Groovy] Class ReportEngine

    • org.simplemes.eframe.reports.ReportEngine

    • Provides low-level access to the external report engine. This is designed to be replaced by a mock engine for testing purposes. The external report engine is not designed to be easily mocked, so this is the place to mock the behavior.

      Logging

      The logging for this class that can be enabled:

      • debug - Parameters and arguments.
