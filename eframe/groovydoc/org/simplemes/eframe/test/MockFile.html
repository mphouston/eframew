Package: org.simplemes.eframe.test

[Groovy] Class MockFile

      • org.simplemes.eframe.test.MockFile 

      • @java.lang.SuppressWarnings(value: UnusedMethodParameter)
class MockFile
extends java.io.File

        A mock File that can be used to write text to string for later use. The file reader will rely on the text written to the mock file by previous calls to the writer.
