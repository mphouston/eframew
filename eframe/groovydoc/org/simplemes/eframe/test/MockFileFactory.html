        • Property Detail

          • private int existCountTrue

            This is the number of times the exists() method should return true. Used to simulate file name collisions.

          • private MockFile lastFileCreated

            The last mock file to be created by this mock factory.

          • private java.util.Map simulatedContents

            The simulated file contents for a simulated file system.

            For Example: 

             FileArchiver.fileFactory.simulatedContents["ref1.arc"] = '...'
            This uses the given string as the contents of the Mock file ref1.arc.

          • private java.util.Map simulatedFiles

            The simulated file system structure.

            For Example: 

             FileArchiver.fileFactory.simulatedFiles = ['../arch'                : ['../arch/2018-03-23'],
                                            '../arch/2018-03-23'     : ['ref1.arc', 'ref2.arc']]
            This simulates a directory (../arch) with a sub-directory (../arch/2018-03-23) that contains two files (ref1, ref2).
