        • @java.lang.SuppressWarnings(value: CatchThrowable) static boolean checkScriptsOnPage(java.lang.String page)

          Checks the given page text for all <script> sections and compiles them in Javascript. The ECMAScript engine throws parsing exceptions when the javascript is badly formed.

          If no <script> sections, then this method does nothing.

          Parameters:
          page - The HTML Page with <script> tags. Must be paired.

        • static java.lang.String extractBlock(java.lang.String javascript, java.lang.String startText)

          Extract a single block of javascript that starts with the given text. Used to find a single block of code that starts with the given text. For example: find the script code that starts with 'function flexTypeEditor(' will find the first occurrence of the string, then finds the first bracket after this text. Then finds the corresponding end bracket and returns all of this text.

          Will fail with an exception if the brackets and quotes are not balanced. Also does not support comments that have quotes or brackets in the comment.

          Supports using square brackets ([]) if the startText ends with [. This will find all of an array's elements.

          Parameters:
          javascript - The full javascript to search.
          startText - The starting text to search from (e.g. 'function flexTypeEditor(').
          Returns:
          The entire block of code, including the start text.

        • static java.lang.String extractProperty(java.lang.String page, java.lang.String name)

          Extracts the given property from a javascript segment. Returns the value. For example, extractProperty('{name: "ABC"}','name') will return "ABC" (without the quotes).

          This is not a very flexible method. It only supports simple quotes and does not allow escaped quotes in the value string.

          Parameters:
          page - The javascript text for the page.
          name - The property name.
          Returns:
          The value (not including quotes).

        • static java.util.List<java.lang.String> extractScriptsFromPage(java.lang.String page)

          Extracts the scripts from the given page text. Finds all <script> sections and returns them as a list of scripts.

          Currently does not handle </script> embedded in a quoted string in the script section. This will cause a parse exception.

          Parameters:
          page - The HTML Page with <script> tags. Must be paired.
          Returns:
          the scripts (list).
