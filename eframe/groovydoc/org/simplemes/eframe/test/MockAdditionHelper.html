[Groovy] Class MockAdditionHelper

    • org.simplemes.eframe.test.MockAdditionHelper
    • All Implemented Interfaces and Traits:
      AutoCleanupMockInterface, org.spockframework.mock.IDefaultResponse

      @groovy.transform.ToString(includePackage: false, includeNames: true)
class MockAdditionHelper
extends java.lang.Object
implements AutoCleanupMockInterface, org.spockframework.mock.IDefaultResponse

      Builds a very simple mock/stub Addition that is found by the getHelper. This class restores the original AdditionHelper.instance during BaseSpecification.cleanup(). 

         new MockHelper(this,TestAddition)

      This will simulate an addition discovered by the framework for use in other areas for test purposes.
