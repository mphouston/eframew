Package org.simplemes.eframe.test

Package org.simplemes.eframe.test Description

Classes to support automated testing - Not for Production Use.
  • Interface Summary 
    Interface Description
    AutoCleanupMockInterface Defines the methods need by a mock object that handles its own cleanup.
  • Class Summary 
    Class Description
    AdditionTestUtils Utility methods for testing additions in unit tests.
    BaseAPISpecification The base class for Spock tests that need to make API calls (over HTTP) to a running (embedded) server.
    BaseDashboardSpecification This is the common Spock specification base class for testing Dashboard related activities.
    BaseFailureListener The failure listener for Spock specifications.
    BaseGUISpecification This is the common Spock specification base class for GUI/GEB tests.
    BaseJSSpecification This is the common Spock specification base class for GUI/GEB tests with support for easy testing fo javascript libraries in a real browser page.
    BaseMarkerSpecification This is a base class that provide additional support for testing the Freemarker marker (helper) classes.
    BaseSpecification The base class for most non-GUI tests.
    BaseWidgetSpecification This is a base class that provide additional support for testing the UI widget classes.
    CRUDGUITester A full-scale GUI tester to test a standardized set of definition pages for a single domain/controller.
    CompilerTestUtils Utility methods for testing annotations and other compiler features in unit tests.
    ControllerTester Helps tests common features of controller classes, such as security and the TaskMenu support.
    DashboardUnitTestUtils Utility methods to help test Dashboard features.
    DataGenerator A simple data generator for test data.
    DomainTester Helps tests common features of domain classes, such as constraints and field order.
    GlobalSpecExtension The global Spock extension used to run the reportFailure() method on the specifications.
    HTMLTestUtils Some utilities for testing generated HTML code.
    InitialDataRecords A holder that records the initial data records that are normally loaded by all initial data loaders.
    JavascriptTestUtils Some utilities for testing generated Javascript code.
    MockAdditionHelper Builds a very simple mock/stub Addition that is found by the getHelper.
    MockAppender A Mock appender for the logback logging system.
    MockBean Builds a very simple mock/stub context and lets the context find the given bean with the getBean() method.
    MockControllerUtils Builds a very simple mock/stub version of ControllerUtils that returns the given controller(s) classes as possible controllers.
    MockDomainUtils Builds a very simple mock/stub version of DomainUtils that returns the given domain(s) classes as possible domains.
    MockFieldDefinitions A mock of the FieldDefinitions, designed to easily create the field definitions needed for marker and related testing.
    MockFieldExtension Builds a very simple mock/stub version of ExtensibleFieldHelper that returns the given custom fields.
    MockFile A mock File that can be used to write text to string for later use.
    MockFileFactory A mock FileFactory that can be used to write text to string for later use.
    MockPersistentProperty A very simple mock/stub of a PersistentProperty for mocked domain use.
    MockPreferenceHolder Builds a very simple mock/stub for the PreferenceHolder that returns the given preference.
    MockPrincipal A mock of the user Principal for use in unit tests only.
    MockRenderer Builds a very simple mock/stub for the view renderer.
    MockSecurityUtils Builds a very simple mock/stub for the SecurityUtils.
    UnitTestRenderer Defines a Freemarker renderer that can render views from memory(string).
