Package: org.simplemes.eframe.test

[Groovy] Class BaseAPISpecification

              • org.simplemes.eframe.test.BaseAPISpecification

              • The base class for Spock tests that need to make API calls (over HTTP) to a running (embedded) server. Supports a current login for API access. This test always starts the embedded server. This also creates an HTTP client that sets followRedirects to false.

                The test should login() for most tests. All later requests will be sent with the login authorization cookie (until logout() is called).

                Note: Most test specs don't need to call logout(). If the user does not change, then the login authorization cookie will be re-used by later tests.

                  • java.lang.String sendRequest(Map options)

                    Sends an HTTP request to the embedded server using the internal client.

                    options

                    The input options for this method are:
                    • uri - The URI to send the request to (Required).
                    • content - The request's entire content (typically JSON or a Map for form submission)(Required for post/put).
                    • method - The request's method (Default: 'get'). Allowed values ('get', 'post', 'put', 'delete').
                    • status - The expected status code from the request (Default: HttpStatus.OK (200) ).
                    • locale - The language locale for the request.
                    If the expected status code is given, then response must have that status code.
                    Parameters:
                    options - The options. Required.
                    Returns:
                    The response as a string
