class WebClientLookup
extends java.lang.Object

      A support class for GUI tests that uses the client sub-module's localization files for lookups.

      Note: Not supported for production.

        • @java.lang.SuppressWarnings(value: unused) static java.lang.String lookup(java.lang.String key, java.util.Locale locale, java.lang.Object[] args)

          Convenience method for general client localization lookup.

          Note: Not supported for production.

          Note: This lookup will flag missing .properties entries.

          Parameters:
          key - The key to lookup. If it starts with '*', then the return value will start with a '*'. The lookup will take place without the '*'. This is used to support required field labels.
          locale - The locale to use for the message. (Default: Request Locale)
          args - The replaceable arguments used by the message (if any).
          Returns:
          The looked up message.
