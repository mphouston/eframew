Package: org.simplemes.eframe.test

[Groovy] Class BaseDashboardSpecification

                • org.simplemes.eframe.test.BaseDashboardSpecification

                • This is the common Spock specification base class for testing Dashboard related activities. It provides a way to define dashboard activities in memory for a specific test to avoid creating dozens of test activities as .ftl files.

                  This base class provides creation of simple one/two panel dashboards with memory-based content and buttons. This allows you to test dashboard functions with minimal dependencies on other elements.

                  This class depends on the main org.simplemes.eframe.dashboard.controller.DashboardController and DashboardTestController to provide the page displays needed for the dashboard itself and the content (test controller).

                  For example: 

                     def page1 = """ ... """
   buildDashboard(defaults: [page1,page2], buttons: [page3,page4])
   displayDashboard()
   clickButton(0)  // Displays page3 in the Panel 'B'.

                  Note: The activity content should be wrapped in tags. If not, it will be treated as HTML test to render in the page.

                    • @java.lang.SuppressWarningsSuppressWarnings DashboardConfig buildDashboard(Map options)

                      Builds a test dashboard for use by this GUI tester. Builds a single or two panel dashboard with dynamic content. Supports buttons to display additional dynamic activities in the last panel.

                      Options

                      • defaults - The default activity content for the panels. (Required: Array of Strings - 1, 3 or 3 elements).
                      • buttons - The buttons activity content for the dashboard. (Optional). Array of Strings or Maps with DashboardButton fields (label, url, panel, title, css, size, buttonID).
                      Note: If the URL's (defaults or buttons values) start with a '/', then it assumed to be a URL, not the activity content.
                      Parameters:
                      options - The options.
                      Returns:
                      The dashboard configuration object.
