Package: org.simplemes.eframe.test

[Groovy] Class BaseGUISpecification

              • org.simplemes.eframe.test.BaseGUISpecification 

              • @groovy.util.logging.Slf4j
class BaseGUISpecification
extends BaseSpecification

                This is the common Spock specification base class for GUI/GEB tests. This includes login/out and debug functions for GEB elements.
