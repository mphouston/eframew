Package: org.simplemes.eframe.preference

[Groovy] Class PreferenceHolder

    • org.simplemes.eframe.preference.PreferenceHolder

    • Defines a user preference object for given user, page and HTML element. This includes a DSL for accessing these preference settings and for persisting them. This object hides most of the mechanics of saving and finding the preferences. See GUI State Persistence in the framework guidelines.

      Example

       PreferenceHolder preference = PreferenceHolder.find {
   page '/app/testPage'
   user 'admin'
   element 'OrderList'
 }

 def columnPref = preference[column] ?: new ColumnPreference(column: column)
 columnPref.width = newSize
 preference.setPreference(columnPref).save()
      Note: You can switch to a different element if needed using: preference.element = 'OrderListB'

      Note: Due the quirks in Groovy DSL processing, do not use values for the values (page, user, element) such as 'id', 'page', 'user' or 'element'. These get set to null if used. Use another variable name such as 'idParam', 'pageParam', 'userParam' or 'elementParam'.

      Logging

      The logging for this class that can be enabled:
      • warn - Warns if a null element is used in the find() call.
      • trace - Logs a stack trace when a null element is used in the find() call.
