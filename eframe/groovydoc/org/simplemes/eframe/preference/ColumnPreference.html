Package: org.simplemes.eframe.preference

[Groovy] Class ColumnPreference

      • org.simplemes.eframe.preference.ColumnPreference

      • This holds the specific preferences for a single column in a list. The column's width, display order and sorting options are stored here.

        User Preference Usage

        This preference is stored in the user preference table under the keys:
        • page - The URI of the page the column is in. Stripped of arguments.
        • user - The user name.
        • element - The HTML ID of the grid this column belongs to.

          • boolean determineIfEmpty()

            Determines if this column preference is empty (has not settings). This is used to keep the list of column preferences as small as possible.

            Note: This is not called isEmpty() to avoid XML parsing issues.
