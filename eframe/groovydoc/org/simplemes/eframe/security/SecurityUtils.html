        • @java.lang.SuppressWarnings(value: GroovyAssignabilityCheck) io.micronaut.http.HttpResponse checkRoleFromSubClass(java.lang.Object controller, java.security.Principal principal)

          Verifies that the sub-class

          Secure:
          annotation is valid for this given user principal. Makes sure the user has at least one of the roles required by the parent controller sub-class.

          Note: This should only be used by controller super-classes.

          This works around an issue with the core Micronaut security not checking the sub-class's

          Secured:
          annotation for methods provided by the parent class.
          returns:
          The forbidden response if the user does not have the correct role.
          Parameters:
          controller - The controller class.
          principal - The user principal.
