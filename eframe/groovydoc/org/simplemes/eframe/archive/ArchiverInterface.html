Package: org.simplemes.eframe.archive

[Groovy] Interface ArchiverInterface

  • Defines the interface for an archiver. Provides ways to remove/restore old records from the database.

    • Method Detail

      • void archive(java.lang.Object object)

        Adds the given object to the archive. This method can be called several times.

        Parameters:
        object - The domain object to archive.

      • void cancel()

        Cancels the archive action. Removes the archive (file) and does not delete the object archived.

      • java.lang.String close()

        Closes the current archive and removes the archived records from the database.

        Returns:
        The archive reference for the saved archive (e.g. a file for the FileArchiver).

      • List findAllArchives()

        Gets a list of archive file references that exist in the active archive area (folder).

        Returns:
        The list of file references.

      • java.lang.String makeArchiveRefBase(java.lang.Object domainObject)

        Build the archive basic reference name. This is a dynamic value that is intended to avoid filling up a directory with .arc files. The object's id can be used with this reference base to make a legal path name.

        This is a relative path name from the archive topFolder location. When used with org.simplemes.eframe.archive.FileArchiver#makePathFromReference(String), the actual file path can be found.

        Parameters:
        domainObject - The domain object to build the archive file name from.
        Returns:
        The base reference.

      • java.lang.Object[] unarchive(java.lang.String ref, java.lang.Boolean save)

        Unarchive the given file and save all of the domain objects in the archive.

        Parameters:
        ref - The file reference provided by the archive process.
        save - If true, then the unarchived records will be automatically saved by the unarchive logic.
        Returns:
        A list of domain objects that were created.
