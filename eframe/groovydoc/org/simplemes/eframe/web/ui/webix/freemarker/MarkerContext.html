[Groovy] Class MarkerContext

    • org.simplemes.eframe.web.ui.webix.freemarker.MarkerContext 

    • @groovy.transform.ToString(includeNames: true, includePackage: false)
class MarkerContext
extends java.lang.Object

      Defines the context a given marker instance is executing in. This is typically used to make sure the marker knows what controller it is associated with.
