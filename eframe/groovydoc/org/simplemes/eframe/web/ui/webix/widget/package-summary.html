Package org.simplemes.eframe.web.ui.webix.widget

Package org.simplemes.eframe.web.ui.webix.widget Description

GUI Widgets to generate UI elements as Toolkit-specific elements. All classes in this package are intended to be moved to a new module someday, when other toolkits are supported.
