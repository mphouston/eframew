Package: org.simplemes.eframe.web.ui.webix.freemarker

[Groovy] Class DashboardMarker

      • org.simplemes.eframe.web.ui.webix.freemarker.DashboardMarker

      • Provides the implementation of the <ef:dashboard/> tag. This tag is used to display configurable dashboard with application and user-defined elements on a page. 

        
 <
        efDashboard:
        category="MANAGER"/>

        Logging

        The logging for this class can be enabled:
        • trace - Logs full details of dashboard settings and panel hierarchy when one is displayed.
