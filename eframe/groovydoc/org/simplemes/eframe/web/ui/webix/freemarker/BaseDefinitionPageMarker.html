[Groovy] Class BaseDefinitionPageMarker

      • org.simplemes.eframe.web.ui.webix.freemarker.BaseDefinitionPageMarker 

      • @groovy.util.logging.Slf4j
abstract class BaseDefinitionPageMarker
extends BaseMarker

        Provides the basic behavior for the efShow, efCreate and efEdit definition page Freemarker markers. This creates the HTML/JS needed to process a single top-level domain object with controls to allow editing and other actions on the domain.
