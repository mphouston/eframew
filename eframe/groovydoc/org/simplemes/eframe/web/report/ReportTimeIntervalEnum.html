[Groovy] Enum ReportTimeIntervalEnum

      • org.simplemes.eframe.web.report.ReportTimeIntervalEnum
      • All Implemented Interfaces and Traits:
        java.lang.Comparable

        final enum ReportTimeIntervalEnum
extends java.lang.Enum<E<E>>
implements java.lang.Comparable

        Defines a report/chart interval of time. The enum is capable of calculating the absolute start/end of the interval based on the current time. This is typically used for filtering report/chart data to specific amount of time. These enumerations are usually evaluated relative to 'now'.
