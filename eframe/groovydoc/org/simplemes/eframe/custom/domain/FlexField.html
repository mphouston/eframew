implements FieldInterface, FieldTrait

      Defines the basic user-defined data field used explicitly in domain class. This defines the basic type of the field and some hints for appearance and validations (list of valid values, etc). This is normally a child of the FlexType to group related items together. The actual values are stored in a a user definable field (flexFields).
