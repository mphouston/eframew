Package: org.simplemes.eframe.custom.domain

[Groovy] Interface FieldInterface


  • interface FieldInterface

    Defines the basic user-defined data field. This defines the basic type of the field and its properties. This is used for custom field extensions on domain objects.

        The format of this field (e.g. String, integer, etc). This should never change on existing records without database migration! It is possible to change to a String format for most cases, but most other changes will require a migration.
