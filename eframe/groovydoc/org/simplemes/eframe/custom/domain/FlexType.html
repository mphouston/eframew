extends java.lang.Object
implements ChoiceListItemInterface, ConfigurableTypeInterface

      Defines the basic user-defined data types. The category can be extended in sub-classes at the application level.

      The actual data values are typically stored in text fields.

        • private java.lang.String category

          The flexible category name. This is used to restrict use of the type to specific areas of the application. Each domain object that uses a FlexType should specify the category.

          A default Category of 'BASIC' is provided by the framework, but it should not be used frequently. (Default:'BASIC'). Required.
