[Groovy] Class FieldGUIExtension

    • org.simplemes.eframe.custom.domain.FieldGUIExtension 

    • @io.micronaut.data.annotation.MappedEntity(value: field_gui_extension)
@DomainEntity
@groovy.transform.EqualsAndHashCode(includes: [domainName])
class FieldGUIExtension
extends java.lang.Object

      This defines the field extensions added to the GUI for a given domain class. This mainly adjusts the order of fields on the standard definition GUIs and lists. This can also add custom fields to those GUIs.
