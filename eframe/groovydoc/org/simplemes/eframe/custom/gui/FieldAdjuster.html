        • static java.util.List<java.lang.String> applyUserAdjustments(java.lang.Class domainClass, java.util.List<java.lang.String> fieldOrder)

          Finds and applies any field order adjustments to the core field order. Will check the given domain class and any super classes.

          Parameters:
          domainClass - The domain class.
          fieldOrder - The current field order array. This modifies the list passed in.
          Returns:
          The effective field order list. This is a single flattened list of field names.

        • static java.util.List<FieldAdjustmentInterface> determineDifferences(java.util.List<java.lang.String> listA, java.util.List<java.lang.String> listB)

          Compares the two display order lists and generates a list of field adjustments needed to make the fields display in the listB order. Tries to use a limited add/remove/move list of changes. Will fall back to a replace all approach if the result does not work.

          Parameters:
          listA - The original list.
          listB - The desired list order.
          Returns:
          The differences needed to achieve the listB ordering.
