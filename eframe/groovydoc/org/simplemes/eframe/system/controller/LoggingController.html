          • @io.micronaut.http.annotation.PostPost Map setLoggingLevel(@io.micronaut.http.annotation.BodyBody java.lang.String body)

            This sets or clears the current logging level for the given logger. Can be used to clear the logging level for a logger so that the level from the parent(s) is used.

            body

            The request body should be a JSON object with these values:
            • logger - The logger name (e.g. full class name)
            • level - The logging level.
            Returns:
            The level settings for this logger.
