Package: org.simplemes.eframe.application

[Groovy] Class ExtensionPointDocumentExtractor

    • org.simplemes.eframe.application.ExtensionPointDocumentExtractor

    • A utility application that will extract the ExtensionPoints defined in the current application's build/classes/main/groovy directory. Used to generate the 'build/generated/extensions.adoc' file for use in your documentation.

      This utility ignores .class files with a $ in the the name.

      Use the gradle task generateExtensionDoc to execute this utility.
