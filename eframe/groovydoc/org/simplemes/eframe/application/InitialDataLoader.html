Package: org.simplemes.eframe.application

[Groovy] Class InitialDataLoader

    • org.simplemes.eframe.application.InitialDataLoader

    • Loads any initial data records needed for a new system. Generally, this just calls each domain's initialDataLoad() method (if defined). This also honors the static field static initialDataLoadAfter = ['org.simplemes.eframe.security.Role'] to make sure this class is loaded after one or more other classes.

      Logging

      The logging for this class that can be enabled:
      • info - Performance timing.
      • debug - Debugging information. Starting message.
      Returns:
      The list of classes loaded, in the order loaded.
